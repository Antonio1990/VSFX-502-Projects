RISPEC NOT FOUND

Technical documents for the RenderMan Interface Specification (RISPEC) are shipped in HTML format with PIXAR's prman/RAT products. Generously they are also made available on-line. Cutter is designed to take advantage of these excellent sources of information. For example, double clipping (+option) on the name of a shading language function, datatype or global variable should open a RISPEC document in Cutters internal web browser.

The information used by Cutter to link to the correct PIXAR HTML page is specified in the prefs.ini file.
